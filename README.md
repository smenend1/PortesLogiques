# Portes Lògiques Plus v5

PWA educativa de portes lògiques.

## Novetat principal
El constructor permet crear circuits amb fins a 8 portes.

Cada porta pot usar:
- entrades A, B, C i D;
- sortides de portes anteriors, per exemple G1, G2, G3...

Això evita circuits circulars i fa possible treballar exercicis de batxillerat amb més portes.
