# Portes Lògiques Plus v4

PWA de laboratori visual de portes lògiques.

## Novetats v4
- Constructor avançat de circuits amb 4 entrades A, B, C i D.
- Tres portes configurables i NOT opcional a A, B, C, D, X, Y, Z i S.
- Logigrama visual del circuit.
- Taula de veritat completa amb valors intermedis X, Y, Z i S.
- Expressió booleana automàtica.
- Simulació pas a pas.
- Biblioteca d'exemples.
- Mode docent i fitxes imprimibles.
