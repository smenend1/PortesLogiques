# Portes Lògiques Plus v4.1

Versió conservadora de revisió i qualitat sobre la v4.

## Canvis principals

- Manté l'arquitectura i les pestanyes de la v4.
- Millora la visibilitat de la sortida final S.
- Millora el logigrama del constructor avançat amb formes de portes més clares.
- Afegeix ajuda ràpida dins del constructor.
- Millora la impressió de taules llargues.
- Millora la diagnosi d'arrencada i la neteja de memòria cau.
- Actualitza el service worker amb una cau nova per evitar versions antigues.
